## About The Project
A package that helps you with everything related to randomness 🔄

### Methods
| Method             | args                               | returns |
| :---               | :-------                           | :-----: |
| 👍 randomBoolean() |                                    | boolean |
| 🔮 randomEmoji()   |                                    | string  |
| 📝 randomString()  | **length**: length of the string   | string  |
| 🔢 randomInt()     | **max**: maximum number            | number  |
